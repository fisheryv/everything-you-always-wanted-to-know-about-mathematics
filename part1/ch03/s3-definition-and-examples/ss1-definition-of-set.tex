\subsection{``集合''的定义}

让我们从定义开始。正如我们上面开解释的那样，我们通常认为集合的特征是分组到该集合中的对象，以及使该分组有意义的属性。以下定义试图使该概念尽可能精确，同时还介绍了一些相关符号和术语。

\begin{definition}
    \textbf{集合}是具有共同明确定义属性的所有对象的整体。集合中的对象称为集合的\textbf{元素}。数学符号 ``$\in$'' 表示短语``是……的元素''（``$\notin$'' 表示``不是……的元素''）。
\end{definition}
